Google Analytics Exclusion

Visiting this page will set a cookie on the visitor's browser. This works in combination with a filter set up in Google Analytics to exclude traffic from anyone visiting this page. It is a persistent cookie, so a person only need visit this page once for the cookie to be effective for future visits.

Technical Details

Setting the cookie

The cookie is set by a combination of:

a) javascript in the body tag that includes: pageTracker._setVar('exclude_visitor')
b) same Google Analytics tracking code used throughout the site inserted at the end of this page.
(View source code for specifics)

The _setVar() value is stored in the _utmv cookie.

GA Exclude Filter:

Filter Type: Custom filter > Exclude
Filter Field: User Defined
Filter Pattern: exclude_visitor (must match the name of the _setVar)
Case Sensitive: No

The value of _setVar() is stored in the Google Analytics field labeled User Defined.

At this time, it looks like filters are not taken into account when calculating Google's Real Time Beta Stats
